# AstraGuard 🔐

AstraGuard is a cybersecurity-focused web application designed for password analysis, cracking simulation, and security testing.

## 🚀 Features
- Password strength analysis
- Wordlist-based password testing
- File upload for password cracking simulation
- Secure backend processing

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Vercel Deployment

This project is developed for **educational and ethical purposes only**.  
Do NOT use it for unauthorized access or illegal activities.
